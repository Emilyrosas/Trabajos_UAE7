## Objetivos del repositorio

Los principales objetivos del repositorio son 2, para entregas de tareas o trabajos relacionados con la unidad 3, y el siguiente objetivo es la experimentación de diferentes códigos.

La entrega de tareas o trabajos me permite aprender y aplicar los conocimientos aprendidos durante la clase. El repositorio permite tener todo en un solo lugar y con esto consigo que esté ordenado.

La experimentación de código me permite descubrir diferentes caminos para un mismo resultado, esto me permite aprender aún más fuera de la universidad. Para esta caso decidí experimentar menos ya que tengo otro repositorio para ese objetivo, pero aún así tuve la posibilidad de hacerlo.


## Organización del repositorio

La organización es muy básica, tengo carpetas para cada una de las tareas, están seriadas para una mejor distribución. Para las tareas empleó el index, style y js, puede variar el nombre pero siempre están presente los 3. Fuera de las carpetas tengo un index, style y js, esto para la experimentación o despliegue de mi página.
